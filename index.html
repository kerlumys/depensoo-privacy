🗑️ Demande de suppression des données

Dépensoo est disponible sous deux formes : application Android via le Google Play Store et Progressive Web App (PWA) accessible sur iPhone/iPad via Safari à l'adresse kerlumys.github.io/depensoo/. Dans les deux cas, l'architecture est identique : « Zero-Server » — aucune donnée financière, personnelle ou d'identification n'est transmise, centralisée ou stockée sur un serveur externe. L'éditeur n'a aucun accès à vos données et n'en détient aucune copie.

Comment supprimer vos données ?

  1. Via l'application (méthode recommandée) :
    Ouvrez l'application, accédez à l'onglet « Confidentialité & RGPD », puis appuyez sur le bouton « Tout supprimer ». L'effet est instantané et irréversible : l'intégralité du stockage local (budgets, dépenses, paramètres, clé de chiffrement) est effacée.
  2. Via votre système Android (version Play Store) :
    Allez dans Paramètres › Applications › Dépensoo › Stockage › « Effacer les données », ou désinstallez simplement l'application. Cela détruira définitivement la clé de chiffrement locale et l'ensemble de vos historiques de dépenses.
  3. Via Safari (version PWA — iPhone/iPad) :
    Allez dans Réglages › Safari › Avancé › Données des sites web › recherchez kerlumys.github.io › « Supprimer ». Vous pouvez également retirer l'app de votre écran d'accueil — cela n'efface pas les données, seule la suppression via Safari ou le bouton in-app est définitive.

Résumé de la politique

✓ Aucune collecte de données ✓ Zéro cookie ✓ Zéro serveur ✓ Zéro compte requis ✓ Chiffrement AES-GCM 256 bits ✗ Connexion bancaire ✗ Synchronisation cloud ✗ Trackers / Analytics

Dépensoo est une application de gestion budgétaire personnelle fonctionnant 100 % localement sur votre appareil. Elle ne collecte, ne transmet et ne partage aucune donnée à caractère personnel avec aucun tiers. L'ensemble de vos données financières est chiffré et reste exclusivement sur votre smartphone.

Données traitées

Dépensoo traite uniquement les données que vous saisissez vous-même et qui restent stockées exclusivement sur votre appareil :

  • Données budgétaires : budgets, transactions, catégories, revenus — chiffrées en AES-GCM 256 bits
  • Hash du mot de passe : PBKDF2-SHA256, 310 000 itérations, sel aléatoire 128 bits — jamais stocké en clair, jamais transmis
  • Métadonnées de session : date de consentement, devise, préférences d'affichage — stockées localement

Aucune donnée n'est transmise à l'éditeur, à Google, ou à tout autre tiers. L'application ne dispose d'aucun accès réseau aux fins de collecte ou de traitement de données utilisateur.

Sécurité technique

  • Chiffrement local : AES-GCM 256 bits via la Web Crypto API (Android et iOS/Safari)
  • Exports JSON : AES-GCM 256 bits — PBKDF2-SHA256, 310 000 itérations, sel aléatoire 128 bits
  • Mot de passe : Jamais stocké en clair — haché via PBKDF2-SHA256 (310 000 itérations, sel aléatoire 128 bits), jamais transmis
  • Stockage : localStorage chiffré — les données sont illisibles sans la clé dérivée du mot de passe
  • Réseau : Aucune requête réseau émise — l'application fonctionne hors ligne
  • Protection brute-force : après 5 saisies consécutives de mot de passe incorrect, toutes les données sont automatiquement et définitivement supprimées de l'appareil sans possibilité de récupération
  • Verrouillage automatique : l'Application se verrouille après 15 minutes d'inactivité et redemande le mot de passe

Stockage local technique — aucun cookie ni tracker

✗ Google Analytics ✗ Facebook Pixel ✗ Cookies tiers ✗ Fingerprinting ✓ Zéro tracker — garanti

Dépensoo n'utilise aucun cookie au sens de la directive ePrivacy 2002/58/CE. L'application utilise uniquement le stockage local de l'appareil (WebStorage) à des fins strictement fonctionnelles, exemptées de consentement cookie en vertu de l'art. 82 de la loi n° 78-17 et des lignes directrices CNIL — délibération n° 2020-091 du 17 septembre 2020.

Aucun SDK de crash reporting, analytics ou publicité n'est intégré.

En raison de l'architecture 100 % locale et chiffrée de Dépensoo, l'éditeur ne détient aucune copie de vos données et ne peut pas intervenir après votre décès.
